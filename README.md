# Touhou Clone (C++/SFML)
A simple Touhou-style bullet hell shooter developed for a 4th-semester University course.

Tech stack: C++, SFML (Graphics/Input), Doxygen for documentation.
Core Focus: Advanced OOP patterns, real-time game loops, and collision systems.

Looking back, this project was a major milestone in my journey. It's where I truly mastered polymorphism, smart pointers, and the importance of interface-based design when managing complex object hierarchies in C++.
