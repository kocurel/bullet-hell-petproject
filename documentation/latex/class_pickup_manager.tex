\doxysubsection{Detailed Description}
Manages the lifecycle, processing, and rendering of various in-\/game pickups. 

This class is responsible for creating new pickups, updating their state, rendering them on the screen, and cleaning up disabled pickups. It uses {\ttfamily std\+::unique\+\_\+ptr} for automatic memory management of pickups and a mutex for thread-\/safe operations, particularly during pickup creation.
