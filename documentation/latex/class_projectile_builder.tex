\doxysubsection{Detailed Description}
\subsubsection*{template$<$\mbox{\hyperlink{concept_derived}{Derived}}$<$ \mbox{\hyperlink{class_projectile}{Projectile}} $>$ T$>$\newline
class Projectile\+Builder$<$ T $>$}
A builder class for constructing \doxylink{class_projectile}{Projectile} objects. 

This class implements the Builder design pattern, allowing for a fluent interface to set various properties of a projectile before constructing it. It is templated to create specific types of projectiles derived from {\ttfamily \doxylink{class_projectile}{Projectile}}. 
\begin{DoxyTemplParams}{Template Parameters}
{\em T} & The concrete type of \doxylink{class_projectile}{Projectile} to build, which must be derived from {\ttfamily \doxylink{class_projectile}{Projectile}}. \\

\end{DoxyTemplParams}
