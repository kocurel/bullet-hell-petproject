\doxysubsection{Detailed Description}
An attack pattern consisting of a series of 6 bigger falling projectiles. 

These projectiles are shot in an upwards spray with an 18-\/degree rotation between each shot in the series. The pattern sets {\ttfamily switch\+\_\+signal\+\_\+} to true when all projectiles in a series have been shot, indicating completion.
