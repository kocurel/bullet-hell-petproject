\doxysubsection{Detailed Description}
An attack pattern where a series of small projectiles are shot in a line. 

This pattern involves shooting a fixed number of projectiles ({\ttfamily shots\+\_\+in\+\_\+series}) with a small {\ttfamily projectile\+\_\+delay} between each shot. After the series is complete, there\textquotesingle{}s a longer {\ttfamily reload\+\_\+time\+\_\+} before the next series can begin. It signals completion of a series via {\ttfamily switch\+\_\+signal\+\_\+}.
