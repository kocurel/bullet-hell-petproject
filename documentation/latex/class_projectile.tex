\doxysubsection{Detailed Description}
Base class for all projectiles in the game. 

Projectiles are objects that move, can deal damage, and have a hitbox for collision detection. This class provides common properties and methods for all projectile types, and declares pure virtual functions that must be implemented by derived classes.
