\doxysubsection{Detailed Description}
Manages the eager loading and retrieval of game texures. \doxylink{class_texture_manager}{Texture\+Manager} is designed as a Singleton to ensure only one instance of it exists globally. 

\begin{DoxyWarning}{Warning}
If a requested texture is not found the \doxylink{class_texture_manager}{Texture\+Manager} returns a reference to the error texture accessable by \doxylink{class_texture_manager_a803f07530048154181ec7854e4c21168}{get\+Error\+Texture()}. 
\end{DoxyWarning}
