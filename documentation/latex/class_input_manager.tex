\doxysubsection{Detailed Description}
The \doxylink{class_input_manager}{Input\+Manager} class handles keyboard input states. 

This class implements the Singleton design pattern to ensure that there is only one instance of the input manager throughout the application. It tracks which keys were just pressed in the current frame

Checks if a key was just pressed in the current frame. 

A key is considered \textquotesingle{}just pressed\textquotesingle{} if it transitioned from not pressed to pressed within the current frame. This state is reset by \doxylink{class_input_manager_a4a4cdc68f99df31ea018b0d837b80fcb}{clear\+Inputs()} at the end of each frame.
