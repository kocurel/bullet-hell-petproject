\doxysubsection{Detailed Description}
An attack pattern consisting of three projectiles shot simultaneously. 

These projectiles are shot with a 30-\/degree rotation between them, forming a fan-\/like spread. The pattern sets {\ttfamily switch\+\_\+signal\+\_\+} to true when the projectiles have been shot, indicating the completion of this attack burst.
