Detailed Description

Parses event definition files and creates corresponding game events. This class is responsible for reading structured event data from files, interpreting it, and using an EventFactory to instantiate and enqueue these events into the game's event system.

void EventParser::load ( unsigned int file_number)

Loads events from a specific event definition file by its index. This method retrieves a file path from the internally stored list of event files based on the given index, and then calls parse() to process its contents. If the file_number is out of bounds, no action is taken.

Parameters
file_number- the zero-based index of the event file to load from the discovered files.
