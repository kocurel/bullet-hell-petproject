Detailed Description

Enemy storage class. Keeps all enemies inside a vector of unique pointers Is responsible for calling process(), render(), receiving new enemies and cleaning up the disabled enemies.

void Enemies::processEnemies ( Player & player,
Projectiles & projectiles )

Main process function. Is to be called each frame by the game controller. It is resposible for calling cleanupEnemies(),checkWave() and calling Enemy::process(Player&) for each non-disabled enemy and checking collisions with player projectiles.

Parameters
player- a reference to the Player object, necessary to pass as a parammeter to Enemy::process()
projectiles- a reference to the Projectiles object, necessary to check for collisions.
