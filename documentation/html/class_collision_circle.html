bool CollisionCircle::checkCollision ( const CollisionCircle & other) const

Checks if the CollisionCircle object is colliding with another CollisionCircle object.

The two circle are colliding if they overlap ie. the distance between them is lower than sum of their radii.

Parameters
other- a const reference to the other CollisionCircle object.
Returns
Returns true if the two circles are colliding, false otherwise.
