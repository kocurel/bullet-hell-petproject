Detailed Description

template<Derived< Projectile > T>
class ProjectileBuilder< T >

A builder class for constructing Projectile objects.

This class implements the Builder design pattern, allowing for a fluent interface to set various properties of a projectile before constructing it. It is templated to create specific types of projectiles derived from Projectile.

Template Parameters
TThe concrete type of Projectile to build, which must be derived from Projectile.
