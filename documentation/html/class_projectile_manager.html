Detailed Description

Manages collections of player and enemy projectiles.

This class is responsible for owning, processing, rendering, and cleaning up both player-fired and enemy-fired projectiles. It implements the IAddProjectile interface to provide a consistent way for other game entities to add new projectiles. It uses separate mutexes for player and enemy projectiles to allow concurrent access from different threads if needed, while maintaining data integrity.

void ProjectileManager::process ( )

Processes all active projectiles.
